Before cars arrive (admin)

  1. Log in as admin → Admin
  2. Inspections — checklist uploaded
  3. Teams — all cars loaded
  4. Competitions — inspections assigned
  5. Users — approve or invite inspectors and team members
  6. Dashboard — confirm teams appear

Inspector workflow (tablet)

  1. Log in → enter email → type 8-digit code (not the email link)
  2. Select competition → team (car #)
  3. Pass / Fail each item (tap again to clear)
  4. Add comment after pass/fail; tap Save
  5. “Live sync on” = other inspectors see updates for this team

Organizer workflow

  1. Dashboard — progress bars and failures
  2. Export PDF — from dashboard or inspector when a team is selected

Team members

  1. Sign up → wait for admin approval
  2. After approval, log in → My team (read-only)
